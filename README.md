# CarSys
基于SpringBoot实现一款选车、资讯等项目
开发工具：Idea/Eclipse

数据库：Mysql 

缓存：Redis

消息中间件：RabbitMQ

项目构建：Maven

版本控制工具：Git+Github

服务器：阿里云EOS 

服务器中间件：Tomcat   集群：Nginx

测试工具：单元测试：测试方法  接口测试：Swagger2、PostMan、Spring MockMVC、Idea 自带的HttpClients 压力测试：jmeter

操作数据库框架：Mybatis/Mybatis-Plus/Spring Data JPA

前端：Vue+jQuery+BootStrap
